# Анализ данных с использованием решающих деревьев

Проект демонстрирует использование решающих деревьев для классификации набора данных Iris.

## Структура проекта

- `data/` - директория для хранения данных
- `notebooks/` - Jupyter notebooks для анализа
- `src/` - исходный код
- `tests/` - тесты

## Установка

1. Создайте виртуальное окружение:
```bash
python -m venv venv
```

2. Активируйте виртуальное окружение:
```bash
# Windows
.\venv\Scripts\activate
# Linux/Mac
source venv/bin/activate
```

3. Установите зависимости:
```bash
pip install -r requirements.txt
```

## Использование

Основной код находится в файле `testing.py`. Запустите его для визуализации решающего дерева.
